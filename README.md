# CMAA Client Follow-Up Demo Skill

An agent skill for building a focused CMAA/APEX demonstration around post-intake client follow-up: an explainable attention queue, public business research, matched success stories, and advisor-approved scheduling.

Install the skill from this repository with a compatible agent skill manager, or copy `skills/cmaa-client-follow-up-demo` into your agent's skills directory.

This repository contains the skill only. It does not contain a deployable application or a license.

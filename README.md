# CMAA Client Follow-Up Demo Skill

An agent-native CMAA/APEX demonstration built from pre-populated JSON. It uses Context.dev for company discovery and browser research for current activity, then places the complete ranked briefing, evidence, success stories, and advisor follow-up outputs in the Documents pane.

Install the skill from this repository with a compatible agent skill manager, or copy `skills/cmaa-client-follow-up-demo` into your agent's skills directory.

This repository contains the skill and its mock-data asset only. It does not contain a deployable application or a license.
